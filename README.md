 🛒 E-Commerce Full Stack Application

A full-stack E-commerce web application built using **React.js** (frontend) and **Django REST Framework** (backend).


 Features

- User Signup & Login (JWT Authentication)
- Product Listing from API
- Add to Cart functionality
- Remove from Cart
- Quantity management
- Total price calculation
- Responsive UI design

---

 Tech Stack

 Frontend
- React.js
- Axios
- HTML5, CSS3

Backend
- Django
- Django REST Framework
- JWT Authentication

### Database
- SQLite (can be extended to MySQL)
